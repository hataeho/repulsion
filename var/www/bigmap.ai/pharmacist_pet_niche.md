## 1. 약사의 동물용 의약품 취급 자격

| 법률 | 내용 |
|------|------|
| **약사법 제2조** | 약사의 업무 범위에 동물용 의약품 포함 (조제·판매) |
| **수의사법 제12조** | 동물 진료는 수의사만 가능 |
| **동물용 의약품 등 취급규칙** | 동물용의약품 판매업 허가 시 약사 가능 |

**결론:** "약사는 동물용 의약품 취급이 가능하다" — **조건부 참**

## 2. 약사가 할 수 있는 것 vs 없는 것

### ✅ 가능
- 동물용 의약품 판매업 허가 후 판매
- 수의사 처방전에 따른 조제
- 동물용 일반의약품(OTC) 판매

### ❌ 불가
- 수의사 처방 없이 전문의약품 조제
- 동물 진단/진료 (수의사법 영역)
- "반려견용입니다"라고 언급하며 약 판매 (진료행위로 간주)

## 3. 회색지대 분석: "강아지 설사약 주세요"

| 시나리오 | 법적 판단 |
|----------|---------|
| 주인이 증상 설명 → 약사가 동물용 약 조제 | ❌ 수의사법 위반 (무면허 진료) |
| 약사가 "수의사 처방전 가져오세요" | ✅ 적법 |
| 약사가 인체용 일반약을 **사람에게** 판매 | ✅ 적법 (주인이 개에게 주는 건 주인 책임) |

> **핵심:** 약사가 "이건 반려견용"이라 말하면 위법, "일반의약품 판매"로 끝나면 적법.

## 4. 수의사 원격 처방 현황

| 구분 | 인간 의료 | 동물 의료 |
|------|----------|----------|
| 원격 진료 | 2025~ 허용 | 원칙적 금지 |
| 근거법 | 의료법 개정 | 수의사법 제12조 |

**회색지대:**

- 농림축산식품부 "반려동물 비대면 진료" 시범사업 검토 중 (2024~2025)

## 5. 💡 비즈니스 모델: "약국 내 화상 수의사 연결"

### 플로우
```
반려견 주인 → 약국 방문
  → 약국 내 화상 단말기로 수의사 연결
  → 카메라로 반려견 보여주기
  → 수의사 진료 + 전자처방전 발행
  → 약사 즉시 조제
  → 주인 수령
```

### 기존 대비 장점

| | 기존 동물병원 | 약국 화상진료 모델 |
|---|---|---|
| 이동 | 강아지 데리고 병원 | 동네 약국 (도보 5분) |
| 대기 | 30분~1시간 | 화상 즉시 연결 |
| 비용 | 진료+약 합산 (비쌈) | 진료비↓ + 약값↓ (분리) |
| 야간/주말 | 응급동물병원만 | 24시 약국 활용 가능 |
| 약사 수익 | 없음 | 조제료 확보 |

### 법적 적법성
- ✅ 화상으로 동물 보여주기 = "직접 진료" 확대 해석 가능
- ✅ 수의사가 처방전 발행 = 적법
- ✅ 약사가 조제 = 적법
- ⚠ 초진 화상 허용 여부 = 시범사업 동향 확인 필요

### 사업화에 필요한 것
1. **수의사 네트워크** — 화상 당직 수의사 풀
2. **약국 내 화상 단말기** — 태블릿 + 고화질 카메라
3. **전자처방전 시스템** 연동
4. **동물용의약품 재고** 확보

## 6. 🚀 확장 가능성: "펫팜(PetPharm)" 플랫폼

> 수의사와 약사를 연결하는 **반려동물 의약분업 플랫폼**

- 반려동물 1,500만 시대 → 충분한 시장 규모
- 인간 의료의 의약분업이 동물에겐 아직 없음 = **제도적 블루오션**
- 약국 체인(올리브영 등)과 제휴 시 전국 확장 가능
- AI 활용 → 동물 약물 상호작용 체크 시스템 연동
