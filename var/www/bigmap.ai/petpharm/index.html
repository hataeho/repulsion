SLIDE 01

🚨 문제: 반려동물 의료의 비효율

현재 상황 문제점
반려동물이 아프면 → 동물병원 진료+약 합산 → 비용 과다 (4~8만 원)
약국은 동물약 취급 안 함 약사의 전문성이 사장됨
야간/주말 → 응급동물병원만 접근성 극히 제한적
경증 (설사, 피부, 구충) 병원 방문이 과잉 대응

💡 약국 24,000개 vs 동물병원 6,000개 — 약국이 4배 더 많은 접점

💡 해결책: 약국 내 화상진료 부스

반려견 주인 → 동네 약국 「화상진료 부스」 → 27" 대형 모니터에 수의사 얼굴이 크게 보임 → HD 웹캠으로 약사가 반려견을 비춰줌 → 수의사 진찰 + 전자처방전 발행 → 약사 즉시 조제 → 주인 수령

SLIDE 06

🚀 단계별 로드맵

Phase 1 — MVP (6개월)

서울 강남/서초 약국 10곳 + 수의사 5명, 경증만, 화상진료 부스

Phase 2 — 확장 (12개월)

  • 전국 주요 도시 약국 100곳
  • 자체 앱 (예약, 처방전 관리, 반려동물 건강기록)
  • 24시 약국 네트워크, AI 증상 트리아지
  • 듀얼 카메라 원격진료 — 모니터 + 스마트폰 동시 활용

Phase 3 — 플랫폼 (24개월)

  • 반려동물 건강 데이터 → AI 진단 보조
  • 펫보험 연계, B2B 약품 공급
  • 스마트 태그 연동 — AirTag/SmartTag 위치추적 + 원격진료 풀패키지

📱 스마트폰 (약사 손에) — 귓속·피부·잇몸·눈 4K 근접촬영
기존 화상진료 듀얼 카메라 방식
웹캠 고정각 → 전체만 보임 스마트폰으로 귓속·피부·구강 클로즈업
수의사 "잘 안 보여요" 4K + 플래시 = 동물병원급 시야
별도 의료기기 필요 스마트폰 = 만능 스코프 (추가 비용 無)

플랫폼별 지원

기능 🍎 Apple (iOS) 🤖 Android
듀얼 카메라 연결 Continuity Camera Wi-Fi Direct / 자체 앱
카메라 성능 iPhone 12MP 4K Galaxy S 시리즈 4K+
화면 미러링 AirPlay SmartView / Miracast

확장 액세서리

  • 스마트폰 + 귀내시경 어댑터 ($10~20) → 이도경(otoscope) 수준
  • 스마트폰 + 피부확대경 렌즈 → 피부질환 정밀 관찰
  • 진료 영상 자동 녹화 → 건강기록 자동 보관
